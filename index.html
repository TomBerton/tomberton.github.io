I'm a research-minded scientist who enjoys solving problems through data analytics and visualization. In my professional career, I apply critical thinking and logical reasoning, and practice good time management, organization, and communication.

Currently, I am a Teaching Assistant and Senior Tutor for Trilogy Education Services - Data Analytics and Visualization certification course, and web specialist for the Austin Community College Biology Department's website.

I earned my Data Analytics and Visualization Certification from Trilogy Education Services through UT Austin in April 2018, and my Software Testing Certification in May 2016 from Austin Community College.

A few of my projects


NOAA Severe Weather Visualization.

This project analyzed severe weather in the United States between 2010-2016 using Leaflet, D3 and Plotly. The website was deployed onto Heroku

Visualization of the GeoJSON Earthquake data.

Using the USGS GeoJSON I mapped the last 7 days of US earthquake data using JavaScript and Leaflet.

Web Scraping for Mars Facts

Using Pandas/Jupyter Notebook, BeautifulSoup, Splinter, Flask and MongoDB I scraped various websites & created a webpage.


Visualization of the weather across the world.

Using Citypy and the OpenWeatherMap API to visualize the weather of 500+ cities across the world.

Sentiment analysis of the Twitter activity using VADER

Using Pandas and Jupyter Notebook I performed a sentiment analysis of the Twitter activity of various news outlets using VADER (Valence Aware Dictionary for sEntiment Reasoning).

Austin 311 call analysis from 2014-2016.

This is a visual analysis of 311 calls by zip code, department and complaint type using Tableau.
