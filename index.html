I'm a research-minded scientist who enjoys solving problems through data analytics and visualization. In my professional career, I apply critical thinking and logical reasoning, and practice good time management, organization, and communication.

Currently, I'm an Adjunct Professor at Austin Community College, where I teach Nutrition and Biology. I also help manage the Biology Department's website.

I earned my Software Testing Certification in May 2016 and will complete my Data Analytics and Visualization Certification at UT Austin in April 2018.

A few of my projects


NOAA Severe Weather Visualization.

This project analyze severe weather in the United States between 2010-2016 using Leaflet, D3 and Plotly. The website was deployed onto Heroku

Visualization of the GeoJSON Earthquake data.

Using the USGS GeoJSON I visualized the last 7 days of US earthquake data with JavaScript and mapped the data with fault lines using Leaflet.

Web Scraping for Mars Facts

Using Jupyter Notebook, BeautifulSoup, Pandas, and Splinter I scraped various websites and created a webpage using Flask and MongoDB.


Analysis and visualization of the weather across the world.

Using Citypy and the OpenWeatherMap API to visualize the weather of 500+ cities across the world.

Sentiment analysis of the Twitter activity using VADER

Using Jupyter Notebook and Pandas I performed a sentiment analysis of the Twitter activity of various news outlets using VADER (Valence Aware Dictionary for sEntiment Reasoning).

Austin 311 call analysis from 2014-2016.

Complaint analysis of 311 calls and correlation with socioeconomic factors in Austin, Texas over 3 years.
